WHAT I LEARNED

CAREER GOALS

인터랙션과 사용자 흐름을 기반으로, 더 복잡한 구조의 사용자 경험을 설계하고 구현하는 것을 목표로 하고 있습니다. 단순한 구현을 넘어 문제를 정의하고, 그에 맞는 구조를 설계하는 개발자로 성장하고자 합니다.

EXPLORING

Three.js

Three.js를 활용해 인터랙션 표현 영역을 확장하고 있으며, WebGL 기반 셰이더를 활용한 시각적 표현과 구조를 실험하고 있습니다. 기존 인터랙션을 넘어, 3D 환경에서의 사용자 경험을 구현하는 방향으로 확장하고 있습니다.

EXPLORING

React

React를 활용해 인터랙션 로직과 UI 구조를 컴포넌트 단위로 분리하며, 상태 흐름을 기반으로 한 구조 설계를 지속적으로 개선하고 있습니다. 단순한 UI 구현을 넘어, 유지보수성과 확장성을 고려한 구조를 만드는 데 집중하고 있습니다.

EXPLORING

TypeScript

TypeScript는 아직 실무에 적용해보지는 않았지만, 인터랙션 로직과 상태 흐름이 복잡해질수록 타입을 통한 안정성과 예측 가능성이 중요하다고 느끼고 있습니다. 현재는 개인 프로젝트를 통해 적용을 준비하고 있으며, 구조적인 코드 설계를 보완하는 방향으로 활용해 나갈 계획입니다.
